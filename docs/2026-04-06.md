# 📅 2026-04-06 Java_Lambda_Stream

### 🏷️ Topic
자바 람다(Lambda)와 스트림(Stream)

### 📝 What I Learned
#### 람다(Lambda) : 코드의 간결화를 위한 함수형 프로그래밍 스타일의 익명 함수.

형식: (매개변수) -> { 실행코드 }

사용 조건 및 특징:

1. 가독성을 위해 코드가 너무 길어지면 안 됨 (일반적으로 10줄 이내 권장).

2. 인터페이스 내에 구현해야 할 추상 메소드가 단 1개만 존재해야 함 (함수형 인터페이스).

3. 로직이 너무 복잡한 경우에는 사용이 제한됨.

단, 인터페이스 내에 default나 static으로 구현된 메소드는 함께 존재 가능.

#### 스트림(Stream): 람다식과 함께 사용되며, 데이터를 컬렉션에서 물 흐르듯이 순서대로 파이프라인 처리함.

.filter(): 특정 조건에 맞는 요소를 걸러냄 (필터링).

.map() : 보통 변환할때 많이 쓰인다

.collect() : 하나의 묶음으로 모아서 (List, Set 등에)담아주는 역할

.forEach(): 요소를 순회하며 출력 등 최종 작업을 수행함.

'::' 은 아주 쉬운 줄임표다. '클래스명::메소드명' 이렇게만 써준다
