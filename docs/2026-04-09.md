# 📅 2026-04-09 Java_Tread_Inner

### 🏷️ Topic
자바 Thread와 내부클래스

### 📝 What I Learned
웹에는 자주 등장하진 않지만 게임개발을 할때 자주 등장하는 Thread 쓰레드를 학습했다.

쓰레드는 한개의 프로세스 안에서 여러개의 기능을 동시에 수행하는 것을 말하며 병렬작업을 통해 시간이 오래 걸리는 작업을 할때 주로 사용한다.

주로 게임 속 움직임과 행동이나 채팅창의 메세지 수신처럼 계속 돌아가야할때 쓴다.

내부 클래스의 핵심은 숨기기다. Outer클래스와 Inner클래스로 이루어져 있다.

내부클래스는 멤버 내부클래스, 정적 내부클래스, 익명의 내부클래스로 이루어죠 있는데 이중 실무에선 익명의 내부클래스가 등장한다.

익명의 내부 클래스는 인터페이스로 딱 한번마 구현해서 써먹어야할 때 자주 쓰이고 요즘 최신 코드에는 람다식으로 많이 대체되어 쓰인다.

그래도 이전에 짜여진 코드들을 유지보수할때 마주칠 가능성도 있으니 형태나 구조 정도는 파악해 두자.

new 인터페이스명(){...} 이런식으로 이루어져 있으며  뒤에는 @Override가 따라붙는다.

전체적으로 내부클래스는 강한결합이 필요하거나 캡슐화의 기능이 필요할때, 익명의 클래스로 한번만 쓰고 버릴 기능을 구현할때 사용한다.
