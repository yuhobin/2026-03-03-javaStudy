# 📅 2026-04-09 Java_Tread_Inner

### 🏷️ Topic
자바 Thread와 내부클래스

### 📝 What I Learned
웹에는 자주 등장하진 않지만 게임개발을 할때 자주 등장하는 Thread 쓰레드를 학습했다.

쓰레드는 한개의 프로세스 안에서 여러개의 기능을 동시에 수행하는 것을 말하며 병렬작업을 통해 시간이 오래 걸리는 작업을 할때 주로 사용한다.

주로 게임 속 움직임과 행동이나 채팅창의 메세지 수신처럼 계속 돌아가야할때 쓴다.

내부 클래스의 핵심은 숨기기다. Outer클래스와 Inner클래스로 이루어져 있다.

강한결합이 필요하거나 캡슐화의 기능이 필요할때, 익명의 클래스로 한번만 쓰고 버릴 기능을 구현할때 사용한다.
