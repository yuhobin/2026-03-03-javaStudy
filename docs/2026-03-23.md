# 📅 2026-03-23 Java_Inheritance

### 🏷️ Topic
객체지향 프로그램의 특징 상속

### 📝 What I Learned
객체지향의 4대 특성 중 하나인  상속을 공부했다. 
상속의 장점은 반복되는 코딩을 줄일 수 있어 간결해지고 유지보수가 편하다.
다만 상속은 높은 연결성으로 인해 Spring을 이용하는 실무에선 잘 쓰이지않고 오히려 (has-a)포함이 많이 쓰인다.
상속을 쓰면서 필요하다면 오버라이딩을 통해 바꿔서 사용할 수 있다.
실제로 지니뮤직 데이터와 멜론뮤직 데이터 수집을 하여 음원차트 창을 만들어봤는데 기본적인 틀이 거의 유사하여 
지니뮤직 음원차트틀만 만들어도 상속을 통해 멜론뮤직 차트도 간편하게 만들 수 있었다.

### 💡 Note
오버라이딩을 위해 상속을 배우긴했으나 사실 오버라이딩은 상속이 필수적인것은 아니고 나중에 익명의 클래스를 통해 사용할 수 있다.
