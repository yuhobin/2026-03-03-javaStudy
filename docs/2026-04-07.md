### 📝 What I Learned
#### IO (Input / Output)

입력 : Input => 프로그램이 외부 데이터를 받는 것

출력 : Output => 프로그램이 외부에 데이터를 보내는 것 


IO 특징 : Stream 기반 : 데이터를 운반하는 통로. 단방향이다.

바이트 스트림(InputStream, OutputStream, FileInputStream, FileOutputStream) / 문자 스트림(Reader, Writer, FileReader, FileWriter) 

/ 보조 스트림(BufferedInputStream, BufferedOutputStream, BufferedReader, BufferedWriter) / 객체 스트림(ObjectInputStream	, ObjectOutputStream)

#### File : file에 대한 정보

목적 : 메모리 저장 => 프로그램 종료시에 자동으로 데이터가 사라진다

파일 정보 확인 

***파일명 : getName() / ***경로+파일명 : getPath() / 경로명 : getParent() / 파일 크기 : length() => long / ***파일 존재 여부 : isFile()
