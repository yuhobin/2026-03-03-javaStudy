# 📅 2026-03-19 Java_OOP_Constructor

### 🏷️ Topic
객체지향프로그램의 생성자

### 📝 What I Learned
생성자의 개념과 특징 학습. 
객체 생성 시 변수 초기화 용도이며 new와 함께 사용됨. 
리턴 타입이없고 클래스명과 이름이 동일하며 오버로딩을 지원함. 
this()는 생성자 간 중복 코드를 줄이고 현재 클래스의 다른 생성자를호출할 때 사용하며, 반드시 생성자 내부의 첫 번째 줄에 작성해야 함.

### 💡 Note
생성자는 리턴 타입이 아예 없다는 점과 클래스명과 항상 동일하다는 점을 기억해서 일반 메소드와 확실히 구분할 것.
특히 클래스에 생성자를 직접 정의하지 않으면 자바가 자동으로 기본 생성자를 넣어준다.
