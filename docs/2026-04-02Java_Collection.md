# 📅 2026-04-02 Java_Collection

### 🏷️ Topic
자바 컬렉션과 제네릭


### 📝 What I Learned
같은 데이터형들을 모아서 한번에 관리하는게 배열이라면 다른 데이터형들까지 모아서 한번에 관리하는것이 컬렉션이다. 
다른 데이터형을 한번에 모으려면 Object로 이것저것 다 받게 되는데 이를 같은것들끼리 걸러서 받게하는게 제네릭이다.
컬렉션 가는곳엔 제네릭이 따라 갈 정도로 둘은 항상 같이 다닌다고 봐도된다.
컬렉션은 배열의 단점이 보완 되어 크기가 자동으로 조절되고 정렬, 검색 기능이 제공된다. 표준화로 개발의 유지보수에 용이하다.
컬렉션은 List Set Map으로 나뉘며 List 에는 세부적으로 그동안 자주보이던 ArrayList, Vector, LinkedList가 있다.


### 💡 Note
자바 개발자라면 죽을때까지 사용하는 컬렉션
List, Set, Map 차이점 알기
ArrayList, Vector, LinkedList 차이점 알기
