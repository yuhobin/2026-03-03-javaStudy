# 📅 2026-03-18 Java_OOP

### 🏷️ Topic

자바 객체지향 프로그래밍과 변수의 종류

### 📝 What I Learned

변수는 선언 위치에 따라 지역 변수(Stack 영역), 인스턴스 변수(Heap 영역), 정적 변수(Method 영역)로 구분되며 각기 다른 메모리 공간과 생명주기를 가진다.

클래스 영역의 변수(인스턴스, 정적)는 자바에 의해 자동으로 초기화되지만, 메소드 내부의 지역 변수는 반드시 직접 초기화해야 사용할 수 있다.

정적 변수는 static 키워드를 사용하여 모든 객체가 데이터를 공유할 수 있게 하며, 이를 통해 공통된 값을 효율적으로 관리한다.

객체지향의 4대 핵심 특징은 캡슐화, 다형성, 상속, 추상화이며 이는 소프트웨어의 재사용성과 유지보수성을 높이는 기둥 역할을 한다.

### 💡 Note

상속은 연결성이 매우 강하므로 협업 시 다른 개발자의 코드에 영향을 줄 수 있음을 인지하고 신중하게 사용해야 하고 

정적 변수는 프로그램 종료 시까지 메모리에 남으므로 데이터 공유 목적에 맞게 꼭 필요한 경우에만 활용해야 한다.
